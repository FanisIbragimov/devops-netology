7. Выполнение команд через ; осуществляется последовательно независимо от результата их выполнения. Выполнение команд через && осуществляется также последовательно, но зависит от результата выполнения предыдущей. В нашем случае после выполнения первой команды возвращается 0 и вторая уже не выполняется.
